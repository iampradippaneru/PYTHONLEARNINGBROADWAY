
Basic git commands for config setup
1. git config --global user.name "pradip_paneru "
2. git config --global user.name #to check 
3. git config --global user.email " pradippneru444@gmail.com"
4. git config --global user.email  #to check

5. git init 
    => Initializes a repository  # it should be used for only one time

6. git status
    => display the current status of files and commits in the git repository

7. git add .
    => adds all the changes to the git repository

8. git commit -m "<commit message>" 
    => Provide a message to your commit

9. git branch -M main
    => Changes the branch name to "main"

10. git branch
    => Displays all the branches and highlight the current branch
